## `Install`

``` bash
npm i --save @ibyar/decorators
```

``` bash
yarn add @ibyar/decorators
```

Create you decorator that is aware of metadata context for [proposal-decorator-metadata](https://github.com/tc39/proposal-decorator-metadata)
